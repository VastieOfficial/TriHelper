# TriHelper
Утилита для работы с TriCache, только на русском, только для Trilib-Spotify. 

Код писался для себя. **Оно убьет ваш диск** (*рано или поздно*, теперь скорее поздно потому что я добавил кэш), так что аккуратнее

Требуются переменные среды TRICACHE, TRICACHE_PREFETCH.

Программа по-сути работает в паре с [uTriLib](https://github.com/VastieOfficial/utrilib), TRICACHE_PREFETCH - путь к папке `prefetch` uTriLib, TRICACHE - путь к папке TRICACHE (тот же что и в uTriLib)

Поддержка не планируется ввиду того что это просто инструмент "на время разработки", не более.
